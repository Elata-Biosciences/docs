## Token Mechanisms

- **Fee Pipeline**
  - App tokens carry an LP-keyed transfer tax (default 1%, max 2%) that applies only when a transfer touches an allowlisted liquidity-pool address.
  - Fees are routed through the protocol fee pipeline: `FeeCollector` → `FeeSwapper` → destinations.
  - App-revenue fee kinds are split between the app's `ContributorSplit` (default 80%) and the protocol treasury (default 20%). These ratios are governance-configurable.
  - Launch fees (10 ELTA per app) are routed 100% to the treasury.
  - For full details, see the [Fee Pipeline](/apps/design/fee-flow) documentation.
- **Bonding Curve Graduation**
  - Each app token trades on a bonding curve denominated in ELTA. When the curve reaches the graduation threshold (42,000 ELTA), liquidity transitions to a DEX pool with a 2-year LP lock.
