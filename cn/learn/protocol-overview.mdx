## 为什么要做 Elata

神经科技应用开发门槛高，商业化更难。Elata 将“发布应用 + 代币经济 + 社区增长”整合在一个协议中，让开发者从上线第一天就能形成可持续激励。

## Elata 提供什么

### 对开发者

- 支付启动费用后即可创建应用代币
- 预留开发者份额并支持自动质押收益
- 通过联合曲线进行公开发行，价格随需求动态变化
- 可扩展赛事、NFT、奖励等机制

### 对用户与社区

- 在应用市场发现新项目
- 通过联合曲线早期参与应用代币
- 质押代币获取协议分配收益
- 锁仓 ELTA（veELTA）参与治理并分享协议级收益

## 核心流程

1. 开发者发布应用并生成应用代币
2. 用户通过联合曲线买卖代币
3. 交易产生手续费并按规则分配
4. 达到阈值后进入更成熟的流动性阶段

## 关键组成

| 组件 | 作用 |
| --- | --- |
| **ELTA** | 协议代币，用于关键交互与生态激励 |
| **veELTA** | 锁仓权益代币，用于治理与收益分配 |
| **应用代币** | 每个应用独立发行的经济单位 |
| **联合曲线** | 提供早期价格发现与流动性机制 |
| **质押金库** | 支持社区参与并领取分配收益 |
