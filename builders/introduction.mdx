## Builder Summary

Elata gives builders a ready-made launch and fee-routing stack:

- app registration and ownership model
- token launch flow with deterministic defaults
- bonding-curve distribution and graduation
- fee accounting and routing to contributor split + treasury

## What You Avoid Building Yourself

- launch contract wiring
- supply allocation logic
- curve lifecycle state machine
- per-app fee bookkeeping

## Protocol Defaults

| Parameter | Value |
|---|---|
| Launch cost | `110 ELTA` |
| App token supply | `10,000,000` |
| Allocation | `50%` curve, `25%` vesting, `25%` ecosystem |
| Graduation target | `42,000 ELTA` |
| LP lock | `730 days` |
