# API Reference

All routes are `GET` requests. Replace path parameters with valid `0x` addresses. `contentId` is the store-scoped premium content ID, not the ERC-721 `tokenId`.

## GET /users/{address}/elta

Returns the user's ELTA balance.

```json
{
  "address": "0x1234...abcd",
  "token": "ELTA",
  "balance": "1000000000000000000",
  "balanceFormatted": "1",
  "decimals": 18
}
```

## GET /users/{address}/veelta

Returns the user's veELTA voting power and lock details.

```json
{
  "address": "0x1234...abcd",
  "token": "veELTA",
  "votingPower": "500000000000000000",
  "lock": {
    "principal": "1000000000000000000",
    "unlockTime": 1735689600,
    "unlockDate": "2024-12-31T00:00:00.000Z"
  }
}
```

## GET /apps/{tokenAddress}/users/{userAddress}/token

Returns liquid and staked balances for one app token.

```json
{
  "app": {
    "tokenAddress": "0xAppToken...",
    "onchainAppId": 42,
    "vaultAddress": "0xVault...",
    "tokenLaunched": true
  },
  "user": {
    "address": "0xUser..."
  },
  "balances": {
    "liquid": {
      "raw": "5000000000000000000",
      "formatted": "5",
      "decimals": 18
    },
    "staked": {
      "raw": "2000000000000000000",
      "formatted": "2",
      "vaultAddress": "0xVault..."
    },
    "totalRaw": "7000000000000000000"
  },
  "currentlyHoldsAppToken": true
}
```

## GET /apps/{tokenAddress}/users/{userAddress}/gate-access

Evaluates the app's saved gate config against live chain state.

```json
{
  "app": {
    "tokenAddress": "0xAppToken...",
    "onchainAppId": 42,
    "hasGate": true
  },
  "user": {
    "address": "0xUser..."
  },
  "checks": [
    {
      "id": "elta",
      "minimum": "1000000000000000000",
      "actual": "2500000000000000000",
      "satisfied": true
    },
    {
      "id": "appToken",
      "minimum": "5000000000000000000",
      "actual": "3000000000000000000",
      "satisfied": false
    }
  ],
  "meetsGate": false
}
```

## Errors

```json
{
  "error": "Invalid address"
}
```

| Status | Meaning |
| --- | --- |
| `400` | Invalid address or parameter |
| `404` | App or premium-content surface not configured |
| `429` | Rate limited |
| `500` | Internal error |
| `503` | Primitive contract not configured |
