## What veELTA Is

veELTA is vote-escrowed ELTA:

- lock ELTA for a fixed duration
- receive non-transferable voting power
- unlock principal after expiry

## Lock Parameters

| Parameter | Value |
|---|---|
| Min lock | 7 days |
| Max lock | 730 days |
| Boost range | 1x to 2x |

## Boost Math

```
boost = 1 + duration / maxDuration
veELTA = lockedAmount * boost
```

## Position Management

- create lock
- increase locked amount
- extend lock end time
- unlock after expiry

## Governance Role

veELTA is used by governance contracts for voting weight and proposal processes.
