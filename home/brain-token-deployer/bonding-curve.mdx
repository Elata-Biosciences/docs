## Overview

App launches use a constant-product bonding curve:

```
k = reserveElta * reserveToken
```

Buy path:

```
newReserveElta = reserveElta + eltaIn
newReserveToken = k / newReserveElta
tokensOut = reserveToken - newReserveToken
```

## Current Defaults

- Launch cost: `110 ELTA` (`10` fee + `100` seed)
- App token supply: `10,000,000`
- Initial curve supply: `5,000,000` (50%)
- Graduation target: `42,000 ELTA`
- LP lock: `730 days`

## Fee Notes

- Base curve fee defaults to `1%`.
- Fee policy is managed in routing pipeline contracts.
