## Brain Token Launcher Flow

The launcher follows the same protocol launch defaults:

1. Pay `110 ELTA` total (`10` creation fee + `100` seed).
2. Deploy app token stack and initialize curve.
3. Sell through bonding curve until graduation.
4. Move liquidity to DEX and lock LP.

## Defaults

- App token supply: `10,000,000`
- Allocation: `50%` curve, `25%` vesting, `25%` ecosystem
- Graduation target: `42,000 ELTA`
- LP lock: `2 years`

## Fees

- Curve buys use configurable trade fee bps (default `1%`).
- Fees route through `FeeCollector` and `FeeSwapper`.
