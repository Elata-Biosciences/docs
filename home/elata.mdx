---
title: "Elata"
---

Elata is a permissionless protocol for launching and operating app-token ecosystems with explicit on-chain economics.

## Core Idea

- register app ownership and contributor split on-chain
- launch an app token with deterministic defaults
- distribute via bonding curve
- route fees through a transparent fee pipeline

## Default Launch Facts

| Parameter | Value |
|---|---|
| Launch cost | `110 ELTA` |
| App token supply | `10,000,000` |
| Launch allocation | `50/25/25` |
| Graduation target | `42,000 ELTA` |
| LP lock | `2 years` |
